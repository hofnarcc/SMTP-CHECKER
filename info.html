Tool Documentation

Setup

To set up the tools, follow these steps:

  1. Ensure you have Python 3.11.8 installed on your machine. You can download it from the official Python website.
  2. Install the required packages by running pip install tk in your command line.
  3. Create a directory to store the tool scripts and any necessary files.

How to Use

There are two main tools available:

  1. COMBO 2 SMTP [TRANSFORMER]
  2. EMAIL SENDER [CHECKER]

1. COMBO 2 SMTP [TRANSFORMER]

Instructions

This tool transforms email and password combinations into SMTP host configurations. To use:

  1. Run the tool by using the START.bat or by executing the code directly python 01_transform_combo_2_smtp.py.
  2. Select an input file containing email/password combinations [COMBOS] in the format:
    3. username:password
    email:password
  3. Choose an option (1 for username:password and 2 for email:password).
  4. Click "Start Transformation" to generate the output file.
  5. The output will be saved in a directory named transformed_accounts_YYYYMMDD_HHMMSS.

2. EMAIL SENDER [CHECKER]

Instructions

This tool sends 1 test email to 1 SMTP Host using the input of the SMTP servers. To use:

  1. Run the tool by uaing the START.bat or by executing python 02_email_sender.py.
  2. Select the input option for credentials (username:password or email:password).
  3. Enter the recipient's email address [JUST INPUT YOUR OWN EMAIL].
  4. Specify the path to the credentials file that contains email/password combinations.
  5. Set the delay between attempts and the number of concurrent threads.
  6. Select the SMTP hosts you wish to use.
  7. Click "Send Emails" to start the process.
  8. You can pause, continue, or stop the sending process at any time.
